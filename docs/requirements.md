# 需求说明

## 目标
构建一个面向国内用户的“电脑主机推荐配置系统”，根据使用场景、预算档位与偏好模式给出配置清单、价格区间与理由。

## 范围
- 仅推荐台式机主机，不包含显示器/键鼠。
- 仅全新配件，不推荐二手或矿卡。
- 覆盖多场景：游戏、办公、开发、设计/剪辑、轻度 AI。

## 约束与偏好
- 显卡品牌主要为 AMD/NVIDIA，允许“无独显”配置用于低负载场景。
- 推荐结果使用价格区间（非精确价）。
- 数据来源主要来自公开网站价格与参数信息（后续由人工维护或半自动导入）。

## 功能需求
- 用户输入：预算档位、使用场景、模式（性价比/性能/静音）、显卡品牌偏好。
- 输出：完整配置清单（CPU/GPU/主板/内存/存储/电源/机箱）、价格区间、配置理由与风险提示。
- 兼容性校验：CPU/主板平台匹配、内存代际匹配、电源功耗估算。

## 非功能需求
- 前端可视化展示，交互简洁。
- 数据与规则可配置，便于后续扩展与维护。
