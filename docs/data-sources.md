# 数据源与维护

## 数据来源
- 国内电商价格：京东、天猫、拼多多等（人工整理或半自动导入）。
- 参数与功耗：厂商规格页面、公开评测与数据库。
- 性能参考：公开基准排行（仅作为参考，不直接参与计算）。

## 维护流程建议
1. 每月更新核心配件价格区间（CPU/GPU/内存/SSD）。
2. 每季度更新平台兼容信息（新平台/新代际）。
3. 对新增硬件给出简短备注（定位、优势、适用场景）。
4. 通过后端同步任务（占位）或手动导入更新数据库。

## 数据表结构
- 见 `data/components.json`（配件库）与 `data/rules.json`（规则库）。
- 后端存储在 `config_store`（JSONB）。
